# Assets Folder

This folder contains app assets like images and icons.

## Required Assets for Expo

The following files should be added for a complete Expo app:

### Icons & Splash Screens (in project root `/assets`)

1. **icon.png** - App icon (1024x1024px)
2. **splash.png** - Splash screen (2048x2048px)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024px)
4. **favicon.png** - Web favicon (48x48px)

## Default Expo Assets

When you run `expo start` for the first time, Expo will use default assets if these files don't exist. You can replace them later with your custom branding.

## Creating Custom Assets

### App Icon
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Content: Center your logo, avoid text near edges

### Splash Screen
- Size: 2048x2048 pixels
- Background: Solid color matching your brand
- Logo: Centered, minimum 40% size

### Quick Tool
Use: https://www.appicon.co/ to generate all required sizes
